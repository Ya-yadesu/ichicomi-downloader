# 一迅社漫画图片复原下载脚本 (ichicomi-downloader) v4.0

一款专为一迅社漫画网站（[ichicomi.com](https://ichicomi.com/)）设计的高清漫画下载与复原油猴脚本。本脚本致力于提供极简、静默、高效的”一键整话下载”体验，并支持将还原后的高清漫画自动打包为 ZIP。

## 🌟 核心功能

*   **⚡ 一键下载整话**：在漫画阅读页面左下角提供”一键下载整话”悬浮按钮，支持静默后台加载。
*   **🧩 4x4 像素完美复原**：自动解密 GigaViewer 混淆切片，使用 Canvas 将原图拼回，同时**保留未参与切分的边缘像素**，还原无损高清大图。
*   **📦 自动 ZIP 打包与命名**：下载完成后自动将整话漫画图片打包成 `.zip` 压缩文件，并根据页面结构自动命名为 `[漫画系列名] - [章节名称].zip`。
*   **⚙️ 自由配置开关**：支持在脚本顶部代码中轻松开启/关闭 ZIP 打包功能（关闭后将依次下载单张图片）。
*   **🎨 自定义导出格式与画质**：支持导出为 `jpeg`、`png`（完全无损但体积大）、`webp`（高压缩）格式，并支持自定义导出画质。
*   **🔒 安全与防封**：内置下载间隔（250ms 频率限制），防止触发限流与反爬机制，直接复用浏览器已登录的会话，安全且免配置。
*   **📡 RSS 自动检查更新**：左下角提供”RSS”和”自动”双开关。开启 RSS 后脚本定时拉取系列 RSS Feed（仅几 KB）检查更新，发现时间窗口内的新话自动下载。比旧版刷新页面方案更轻量安全，可放心设置高频轮询。
*   **🎯 右键切换预设**：右键 RSS 按钮切换轮询间隔（30秒/1分/5分/10分/30分/60分）；右键自动按钮切换时间窗口（1时/6时/12时/24时/48时/7天）。
*   **📊 RSS 进度条**：RSS 按钮显示渐变动画进度条，走满一次 = 一轮轮询完成，速度随设定自动调整。
*   **🔄 下载重试**：单页下载失败后自动等待 1 秒重试一次。
*   **🛡️ 重复防护**：自动下载静默跳过已下载章节；手动下载弹窗提醒但不阻止。
*   **🧹 自动清理**：超过 15 天的下载记录自动清除。

## 🛠️ 如何使用

1.  **安装脚本**：
    *   确保您的浏览器已安装 **Tampermonkey** 或 **Violentmonkey** 等脚本管理器插件。
    *   点击本页面上方的 **”安装此脚本”** 按钮进行安装。
2.  **开始下载**：
    *   打开一迅社漫画（[ichicomi.com](https://ichicomi.com/)）任意章节的阅读页面。
    *   页面加载完成后，左下角会浮现三个悬浮按钮：**”一键下载整话”**、**”自动：关”** 和 **”RSS：关”**。
    *   点击 **”一键下载整话”** 手动下载当前章节，按钮实时更新下载进度（如 `下载中 (5/20)`）。
    *   **右键** 点击 **”RSS”** 按钮切换轮询间隔，**左键** 开关轮询。
    *   **右键** 点击 **”自动”** 按钮切换时间窗口，**左键** 开关自动下载。
    *   开启 RSS 轮询后脚本定时检查更新；开启自动下载后，发现时间窗口内的新话自动下载。
    *   在非章节页（系列列表、首页等）按钮会自动变灰禁用。
3.  **获取漫画**：
    *   全部下载并复原完成后，按钮会显示 `正在打包 ZIP...`。
    *   完成后自动弹出浏览器下载，保存打包好的漫画 ZIP 压缩文件。

## ⚙️ 进阶配置（可选）

如果需要调整画质、格式或关闭 ZIP 打包，点击油猴插件的”编辑”脚本按钮，修改脚本前部的 `CONFIG` 块：

```javascript
const CONFIG = {
    format: 'image/jpeg',  // 导出格式: 'image/jpeg' / 'image/png' / 'image/webp'
    quality: 0.98,         // 导出质量 (仅 jpeg/webp，范围 0.0 ~ 1.0)
    zipEnabled: true,      // 是否自动打包 ZIP (true: 打包, false: 单张下载)
};
```

> 轮询间隔和时间窗口建议直接通过按钮右键切换，无需编辑代码。
