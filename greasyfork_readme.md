# 一迅社漫画图片复原下载脚本 (ichicomi-downloader)

一款专为一迅社漫画网站（[ichicomi.com](https://ichicomi.com/)）设计的高清漫画下载与复原油猴脚本。本脚本致力于提供极简、静默、高效的“一键整话下载”体验，并支持将还原后的高清漫画自动打包为 ZIP。

## 🌟 核心功能

*   **⚡ 一键下载整话**：在漫画阅读页面左下角提供“一键下载整话”悬浮按钮，支持静默后台加载。
*   **🧩 4x4 像素完美复原**：自动解密 GigaViewer 混淆切片，使用 Canvas 将原图拼回，同时**保留未参与切分的边缘像素**，还原无损高清大图。
*   **📦 自动 ZIP 打包与命名**：下载完成后自动将整话漫画图片打包成 `.zip` 压缩文件，并根据页面结构自动命名为 `[漫画系列名] - [章节名称].zip`。
*   **⚙️ 自由配置开关**：支持在脚本顶部代码中轻松开启/关闭 ZIP 打包功能（关闭后将依次下载单张图片）。
*   **🎨 自定义导出格式与画质**：支持导出为 `jpeg`、`png`（完全无损但体积大）、`webp`（高压缩）格式，并支持自定义导出画质。
*   **🔒 安全与防封**：内置下载间隔（250ms 频率限制），防止触发限流与反爬机制，直接复用浏览器已登录的会话，安全且免配置。
*   **📡 RSS 自动检查更新**：左下角提供"RSS轮询"和"自动检查"双开关。开启 RSS 轮询后脚本定时拉取系列 RSS Feed（仅几 KB）检查更新，发现近 24 小时内发布的新话自动下载。比旧版刷新页面方案更轻量安全，可放心设置高频轮询。

## 🛠️ 如何使用

1.  **安装脚本**：
    *   确保您的浏览器已安装 **Tampermonkey** 或 **Violentmonkey** 等脚本管理器插件。
    *   点击本页面上方的 **“安装此脚本”** 按钮进行安装。
2.  **开始下载**：
    *   打开一迅社漫画（[ichicomi.com](https://ichicomi.com/)）任意章节的阅读页面。
    *   页面加载完成后，左下角会浮现三个悬浮按钮：**”一键下载整话”**、**”自动检查”** 和 **”RSS轮询”**。
    *   点击 **”一键下载整话”** 即可手动下载当前章节，按钮将实时更新下载进度（如 `下载中 (5/20)`）。
    *   打开 **”RSS轮询”** 开关后，脚本会每分钟查询系列 RSS Feed 检查更新；打开 **”自动检查”** 后，发现新话会自动下载。
3.  **获取漫画**：
    *   全部下载并复原完成后，按钮会显示 `正在打包 ZIP...`。
    *   完成后会自动弹出浏览器下载，保存打包好的漫画 ZIP 压缩文件。

## ⚙️ 进阶配置（可选）

如果您需要调整画质、格式或关闭 ZIP 打包，可以点击油猴插件的“编辑”脚本按钮，修改脚本前部的 `CONFIG` 块：

```javascript
// ==========================================
// 脚本自定义配置项（在此调整下载画质 and 格式）
// ==========================================
const CONFIG = {
    format: 'image/jpeg',  // 导出格式可选: 'image/jpeg' (默认), 'image/png' (完全无损但体积大), 'image/webp' (高压缩)
    quality: 0.98,         // 导出质量 (仅对 jpeg 和 webp 生效，范围 0.0 ~ 1.0)
    zipEnabled: true,      // 是否开启自动打包为 ZIP 文件 (true: 开启打包, false: 依次下载单张图片)
    autoCheckIntervalMs: 1 * 60 * 1000,        // RSS 轮询间隔（分钟 * 秒 * 毫秒），默认 1 分钟
    autoRecentWindowMs: 24 * 60 * 60 * 1000   // 自动下载的时间窗口（小时 * 分钟 * 秒 * 毫秒），仅下载近 24 小时内更新的新话
};
```
